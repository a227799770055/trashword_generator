## 幹話產生器
該網站利用 node.js、express 以及 handlebars 完成

## 環境建置需求
* 執行環境： node.js v10.15.0
* 框架：express 
* 模板引擎： express-handlebars

## 安裝與下載
#### 專案下載
    git clone https://github.com/a227799770055/trashword_generator
#### 安裝 package
    npm install
#### 使用 nodemon 啟動伺服器
    npm run dev
